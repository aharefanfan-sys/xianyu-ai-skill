## 输出格式

请生成以下 JSON 格式：

```json
{
    "style": "拍摄风格（专业摄影棚/生活场景/平铺/随手拍）",
    "angle": "拍摄角度（正面/侧面/45度/俯拍/特写）",
    "background": "背景类型（纯色/渐变/实景/模糊背景）",
    "lighting": "光线情况（柔光/硬光/自然光/暗调）",
    "composition": "构图特点（居中/三分法/留白）",
    "text_overlay": "文字水印（价格标签/水印logo/无文字）",
    "image_quality": "图片质量（高清/普通/模糊）",
    "highlights": "突出特点",
    "suggestions": "改进建议"
}
```

## 分析维度

1. **风格**：专业摄影棚 / 生活场景 / 平铺展示 / 随手拍
2. **角度**：正面 / 侧面 / 45度 / 俯拍 / 特写
3. **背景**：纯色 / 渐变 / 实景 / 模糊背景
4. **光线**：柔光 / 硬光 / 自然光 / 暗调
5. **构图**：居中 / 三分法 / 留白
6. **文字**：价格标签 / 水印 / 无

## 注意事项

- 图片路径：{image_path}
- 如果无法分析图片，返回错误信息
