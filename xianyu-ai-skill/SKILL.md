# 闲鱼 AI 商品发布助手

## 功能概览

- 📥 收集租赁设备信息（机型、租金、配置）
- 🧠 学习爆款文案特征（支持上传 + 自动分析）
- 🖼️ 学习爆款商品图片风格（角度、背景、构图）
- ✨ AI 生成符合闲鱼流量规则的商品文案
- 📤 集成闲鱼开放平台 API 发布
- 🔄 用户反馈持续迭代优化

## 使用场景

当用户需要：
- 生成闲鱼商品文案
- 上传/学习爆款文案
- 上传/学习爆款商品图片
- 发布租赁设备到闲鱼

## 工作流程

1. **信息收集** → 引导用户输入设备信息
2. **特征匹配** → 从文案库选取相似爆款 + 图片风格
3. **AI 生成** → 调用 MiniMax 生成文案（参考爆款风格）
4. **规则校验** → 检查是否符合流量规则
5. **预览确认** → 用户确认或修改
6. **发布/保存** → 发布到闲鱼或保存草稿

## 使用示例

```
用户：帮我发一个租赁相机到闲鱼
助手：[启动 Skill]
   - 请提供设备信息：品牌型号、日租金、押金、配置
   - 请上传爆款商品图片（可选）
   - AI 生成文案
   - 用户确认
   - 发布

用户：上传这个爆款链接的图片
助手：[分析图片特征并存储]
```

## 注意事项

- API Key 优先读取环境变量 `MINIMAX_API_KEY`
- 图片支持 JPG/PNG，建议 800x800 以上
- 生成的文案需符合闲鱼流量规则
